France Dataviz, c'est quoi ?

C'est juste une petite application qui te permet de visualiser ton fichier CSV avec les colonnes suivantes :

  • label : Libellé qui va s'affichier quand l'utilisateur clique sur le rond rouge.
  • zipcode : Code postal pour déterminer le lieu ou positionner le rond rouge.
  • latitude : (optionel) Latitude si pas de zipcode.
  • longitude : (optionel) Longitude si pas de zipcode.

Si tu indiques le zipcode (code postal), je retrouve tout seul la localisation. Pour les zipccodes étrangers, faut mettre le préfixe du pays avant. Exemple B- pour la Belgique, CH- pour la suisse, etc.

Alors bien sûr, c'est à toi de faire ton propre fichier CSV. Je peux pas faire grand chose sur ce coup là.

Juste un truc, tu peux donner un titre à ton fichier avec une ligne de commentaire comme ceci : 

    # title=Ceci est le titre de mon fichier

Exemple

    # title=Les endroits ou j'ai passé du temps en Vacances
    label,zipcode
    Mon endroit préféré,05000
    Mon autre endroit préféré,44350
    Manger des frites mayo et de la bière,B-1000

Comment je charge mon fichier

Fichier sur ton ordi, à visualiser juste pour toi

Tu peux cliquer sur le bouton en haut à droite. Il te demande la localisation de ton fichier, et hop il se télécharge, mais juste sur le navigateur. Rien ne va sur un serveur, ta vie privée reste chez toi. Si tu veux t'en assurez, tu peux regarder le code source de cette appli.

Fichier sur un serveur, à partage avec les autres

C'est pas encore fait, mais je prévois de faire un bouton qui permette de partager une dataviz. A toi de mettre ta dataviz en public (avec le fichier CSV posé dans un cloud Google truc, Amazon machin, ou Micromou tagada, FTP, etc.). Pour contourner les problèmes de CORS, il faut mettre une extension '.txt' à ton fichier, même si le contenu est du CSV.
